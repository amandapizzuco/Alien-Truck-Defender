# Alien Truck Defender

Alien Truck Defender is a Pygame arcade shooter game where the player controls a truck and fights waves of aliens.

## Features

- Multiple truck types
- Colour-changing Trucks
- Alien enemies and boss fights
- Sound effects and animations
- Lives system
- Increasing difficulty
- Power-ups and shooting mechanics

## Requirements

Before running the game, make sure you have:

- Python 3 installed
- Pygame installed

## Installation

Install Pygame using:

```bash
pip install pygame
```

## How to Run the Game

Run the Python file using:

```bash
python main.py
```
